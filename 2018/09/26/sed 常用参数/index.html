sed 参数说明

1. 概述

sed 是一种流编辑器，它是文本处理中非常中的工具，能够完美的配合正则表达式使用主要用来自动编辑一个或多个文件

2. 参数说明

命令格式：
sed [-Ealn] command [file …]
sed [-Ealn] [-e command] [-f command_file] [-i extension] [file …]

[-Ealn]

  • -E
  • -a
  • -e command: 直接在指令列模式上进行 sed 的动作编辑(command 为命令行指令)
  • -f command_file: 直接将 sed 的动作写在一个文件内，-f filename 则可以执行 filename 内的sed命令
  • -i extension: 直接修改读取的文件内容，而不是由屏幕输出
  • -l
  • -n: 使用安静 silent 模式，在 sed 的用法中，所有来自 stdin 的内容一般都会被列出到屏幕上，但如果加上 -n 参数后，则只有经过 sed 特殊处理的那一行(或者动作)才会被列出来
  • -r sed 命令支持扩展的正则表达式

Functions

  • a: 在读取下一个输入行之前将 Text 变量放在输出中
  • b[label]: 转移到具有 label 变量的 : 命令, 如果 label 变量为空，那么转移到脚本的结尾
  • c: 删除模式空间
  • d: 删除模式空间，然后开始下一个周期
  • D: 删除通过第一个换行符的模式空间的初始段，然后开始下一个周期
  • g: 保留空间的内容替换模式空间的内容
  • G: 将保留空间的内容附加到模式空间
  • h: 用模式空间的内容替换保留空间的内容
  • H: 将模式空间的内容附加到保留空间
  • i: 将下一行读到模式空间之前将 Text 变量写到标准输出
  • l: 将模式空间写到标准输出，将不可显示字符以 4 位十六进制值显示
    • backslash \
    • alert \a
    • form-feed \f
    • carriage-return \r
    • tab \t
    • vertical tab \v
  • n: 如果不禁止缺省的输出，那么将模式空间写到标准输出
  • N: 使用嵌入式换行符（更改当前的行号）将输入的下一行附加到模式空间, 可以用它来搜索分割成两行的模式
  • p: 将模式空间写到标准输出
  • P: 删除通过第一个换行符的模式空间的初始段写到标准输出
  • q: 转移到脚本的结尾
  • r file: 读取 RFile 变量的内容，在读取下一输入行之前将内容放置到输出
  • s/regular expression/replacement/flags: 替换(结合如下 function 使用)：
    • N
    • g
    • p
    • w file
  • t: 如果自从读取执行 t 子命令的输入行以来做了任何替代，那么转移到该脚本文件中的 :label 变量。如果未指定 label 变量，那么控制转移到脚本的结尾。
  • w file: 将模式空间附加到文件中
  • x: 交换模式空间和保留空间的内容

  • y/string1/string2/: 转换字符

  • ‘!’: 列出非匹配行数据

  • ‘#’: 把注释扩展到下一个换行符以前
  • ‘:label’: 标记 b 和 t 子命令所引用的分支点
  • ‘=’: 打印当前行号码
