netstat 参数说明

netstat - 打印网络连接、路由表、连接的数据统计、伪装连接以及广播域成员

参数说明

  • netstat -a：列出所有当前的连接
  • netstat -at： -t 列出 TCP 协议的连接
  • netstat -au： -u 列出 UDP 协议的连接
  • netstat -ant：-n 禁用域名解析功能
  • netstat -tnl： -l 列出正在监听的套接字
  • netstat -nlpt：查看进程信息
  • netstat -ltpe：-ep 查看进程名和用户名
  • netstat -s：列出所有网络包的统计情况
  • netstat -rn：-r 选项打印内核路由信息
  • netstat -i： 打印网络接口信息
  • netstat -ie：同上，与ifconfig类似
  • netstat -ct：-c 选项持续输出信息
  • netstat -g：-g 会输出 IPv4 和 IPv6 的多播组信息
